# Maran Farms Full Stack

## Frontend
- Location: `frontend`
- Vite + React + TypeScript + Tailwind

## Backend
- Location: `backend`
- Node.js + Express + MongoDB + Razorpay

## Quick Start
1. Setup backend env from `backend/.env.example`
2. Run backend:
   - `cd backend`
   - `npm install`
   - `npm run seed`
   - `npm run dev`
3. Setup frontend env from `frontend/.env.example`
4. Run frontend:
   - `cd frontend`
   - `npm install`
   - `npm run dev`

See `backend/README.md` for complete backend/payment setup details.
